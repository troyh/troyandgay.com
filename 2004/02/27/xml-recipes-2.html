XML recipes

I’m not sure if Recipezaar is the only recipe site to support RSS, but it’s definitely the only recipe site that also supports XML recipes (in RecipeML format and CookML format) and RecipeRPC. FWIW, I don’t think there is another recipe site that publishes 100 recipes a day that supports RSS.
